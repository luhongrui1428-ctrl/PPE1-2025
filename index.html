Programmation et projet encadré : Mini-projet

Accueil – Présentation du projet

Présentation

Le mini-projet s’inscrit dans notre préparation au projet encadré. Il consiste à analyser un ensemble d’URLs en utilisant un script Bash. Le script récupère pour chaque URL : le code HTTP, l’encodage de la page et le nombre total de mots. Les résultats sont ensuite exportés sous forme d'une page HTML contenant un tableau.


Étapes

  • Découverte de la structure d’une page HTML et des éléments de base.
  • Compréhension de l’utilisation de curl et lynx pour récupérer le contenu d’une page web et extraire les informations essentielles.
  • Exercices pratiques pour apprendre à écrire un script Bash récupérant plusieurs informations à partir d’une liste d’URLs.
  • Apprentissage de la création d’un fichier HTML et de la génération automatique d’un tableau via un script Bash.
  • Introduction au CSS et utilisation de Bulma pour améliorer l’apparence des pages générées.
